# 学习记录

## 初始化

1.设置npm淘宝镜像，加速下载速度

```shell
npm config set registry http://registry.npm.taobao.org/
```

 2.初始化项目信息

```shell
npx create-react-app react-app-demo /** react-app-demo 是项目的文件名，可以随意修改*/
```

3.启动项目

```shell
cd react-app-demo
yarn start
```

4.启动成功后会使用3000端口，进入网址即可看到项目预览信息
